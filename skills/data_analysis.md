# Data Analysis Skill

This skill helps you analyze data, provide insights, create visualizations, and generate statistical summaries.

## When to Use

Use this skill when:
- User asks to analyze data
- User mentions "analyze", "statistics", "insights"
- User wants data visualizations
- User asks for trends or patterns
- User needs data summaries

## How to Use

1. Read the data file using `read_file`
2. Parse and analyze the data:
   - Calculate basic statistics (mean, median, mode, std dev)
   - Identify trends and patterns
   - Find outliers and anomalies
   - Perform correlation analysis
3. Create visualizations:
   - Charts and graphs
   - Tables and summaries
   - Heatmaps and scatter plots
4. Provide insights and recommendations
5. Save results using `write_file` if needed

## Analysis Types

### Descriptive Statistics
- Mean, median, mode
- Standard deviation, variance
- Min, max, range
- Percentiles, quartiles
- Frequency distributions

### Trend Analysis
- Time series analysis
- Moving averages
- Seasonal patterns
- Growth rates
- Trend lines

### Correlation Analysis
- Pearson correlation
- Spearman correlation
- Scatter plots
- Regression analysis

### Comparative Analysis
- Group comparisons
- A/B testing
- Before/after analysis
- Benchmarking

## Visualization Types

### Charts
- Bar charts
- Line charts
- Pie charts
- Scatter plots
- Histograms

### Tables
- Summary tables
- Pivot tables
- Cross-tabulations
- Ranking tables

### Advanced
- Heatmaps
- Box plots
- Violin plots
- Treemaps

## Example Response

When analyzing data, structure your response as follows:

1. **Data Overview**: Brief description of the data
2. **Key Statistics**: Main statistical findings
3. **Trends and Patterns**: Identified trends and patterns
4. **Insights**: Key insights and observations
5. **Recommendations**: Actionable recommendations
6. **Visualizations**: Charts and graphs (if applicable)
7. **Limitations**: Any limitations or assumptions
